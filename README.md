This is a project that will analyze data from NASA turbofan degredation dataset to determine remaining useful life (RUL) of an engine.

The data can be downloaded here: [Jet Engine Simulated Data](https://data.nasa.gov/docs/legacy/CMAPSSData.zip)

Information on the different datasets is explained here: [Jet Engine Simulated Data Information](https://data.nasa.gov/dataset/cmapss-jet-engine-simulated-data)

There are train, test sets, and true RUL values for the test examples
